Window Terminal配置

mingzaily / 2021-06-10

场景

最近发现一个 Shell 利器，Windows Terminal，可以在微软商店自行下载

Git-Bash

前置要求是安装好 Git，比如作者的 Git 目录为D:\Git

配置

  • 点击设置

  • 点击左下角，新增，会出现一个默认配置，可以选择 json 编辑，也可以图形化配置

  • 名称自由编辑

  • 命令行，填写 Git 安装目录下 bin 目录的 bin.exe，比如D:\Git\bin\bash.exe

  • 启动目录，%USERPROFILE%即可

总体算完成了

补充

细心的读者会发现部分命令比如ll在 git-bash 可以使用，但是在 Terminal 下的 bash 不能使用 需要我们自行添加

# 编辑Git安装目录D:\Git\vendor\etc下的bash.bashrc文件
# 新增
alias ll="ls -l"

保存即可
