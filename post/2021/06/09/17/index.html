Linux 安装 Golang

mingzaily / 2021-06-09

场景

使用yum或apt-get命令安装，版本不好控制，使用源码安装

源码安装

  • 查看https://studygolang.com/dl所需安装版本的地址

  • 下载

wget https://studygolang.com/dl/golang/go1.15.13.linux-amd64.tar.gz
  • 解压
tar -zxvf go1.15.13.linux-amd64.tar.gz
  • 移动
mv go /usr/local/go
  • 添加环境变量
# 打开
vim /etc/profile
# 添加
export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin
# 运行
source /etc/profile
  • 安装完成
# 查看安装的go版本，校验通过，安装成功
go version
